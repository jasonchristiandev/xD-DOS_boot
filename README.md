# Extended Drive - Disk Operating System (xD-DOS) Bootloader

xD-DOS Boot is a bootloader created **specifically** for xD-DOS. It **does not target any other operating systems** except xD-DOS itself.

This project should not be compiled by itself, but rather as a subrepo of xD-DOS.
